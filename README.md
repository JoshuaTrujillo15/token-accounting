# token-accounting

Accounting for ERC20 tokens and Superfluid Super Tokens

---

## Set Up Project Locally

Clone

```bash
git clone https://github.com/JoshuaTrujillo15/token-accounting.git
```

Install Deps

```bash
yarn install
```

Build and Run

```bash
# lints, compiles, and runs
yarn start
```

Projcet should run on port 5000
